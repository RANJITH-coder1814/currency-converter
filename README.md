# 🎵 Music Player Web App

A simple **Music Player** built using **HTML, CSS, and JavaScript** that allows users to play, pause, and control songs directly in the browser.

## 🚀 Features

* Play and pause music
* Next and previous song
* Song progress bar
* Display song title
* Clean and responsive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```
music-player-webapp/
│
├── index.html
├── style.css
├── script.js
├── songs/
│   ├── song1.mp3
│   ├── song2.mp3
│   └── song3.mp3
└── README.md
```

## ▶️ How to Run

1. Clone the repository
2. Add your music files inside the **songs** folder
3. Open `index.html` in your browser
